Caracteristicas y inconvenientes de PHP

PHP es un lenguaje de programación creado por Ramsus Lerdof. A continuación se van a citar algunas de las caracterisiticas y inconveniesntes de PHP.

Caracteristicas principales de PHP

  • PHP es un lenguaje de programación interpretado.
  • PHP está orientado a objetos, por lo que permite:
    • Abstracción.
    • Modularidad.
    • Encapsulamiento.
    • Polimorfismo.
    • Herencia.
  • PHP es ejecutado en un servidor y el resultado HTML del programa es enviado al buscador web, por lo que el código PHP es invisible al cliente y al buscador.
  • PHP es capaz de conectarse a la mayoría de motores de bases de datos.
  • Al igual que python y javascript, PHP puede expandir su funcionaldad con el uso de módulos.
  • Es software libre.
  • Permite programación orientada a objetos.
  • PHP es un lenguaje de tipado debil, por lo que no requiere que las variables tengan tipo.
  • Tiene manejo de excpeciones.

Inconvenietes de PHP

  • Al ser interpretado en ejecución, puede ser un inconveniente que el código no se pueda ocultar.
  • Al ser un lenguaje interpretado es considerablemente más lento que lenguajes compilados como C o java.
  • Programar en una versión de PHP previa a la 7 con un IDE puede ser complicado ya que las variables no son tipificadas.
